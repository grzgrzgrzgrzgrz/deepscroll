# Code of Conduct

## Expectations

This project expects contributors to:

- be respectful and precise in technical discussion
- assume good intent and challenge ideas without attacking people
- keep feedback actionable, specific, and relevant to the work
- avoid harassment, discrimination, intimidation, and personal attacks

## Scope

This code of conduct applies to project spaces, including issues, pull requests,
code review, discussions, and other public community channels tied to `deepscroll`.

## Reporting

If you experience or witness unacceptable behavior, report it to:

- `grzegorzolszowka@gmail.com`

Reports will be reviewed and handled with discretion.
